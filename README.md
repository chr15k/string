# String helpers

## Usage

```php
echo str_plural('bus');
// output: buses

echo str_singular('buses');
// output: bus

echo str_snake('Hello there');
// output: hello_there

echo str_camel('Hello there');
// output: helloThere

echo str_slug('we wish you a merry christmas');
// output: we-wish-you-a-merry-christmas
```
